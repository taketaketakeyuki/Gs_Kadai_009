# Gs_Kadai_009

## 課題内容（どんな作品か） 
- ログイン機能を付与した上で、非ログイン者向けのサンプルページ（これも一応DBから取得する処理）を作りました。

## 工夫した点・こだわった点
- ログインページからサンプルページを見られるように
- サンプルページから新規登録・ログインできるように
- サンプルページの表示処理も、本来sessionIDを用いるところを活用して操作しました。

## 質問・疑問（あれば）
